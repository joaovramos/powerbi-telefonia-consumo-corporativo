# Telefonia — Análise de Consumo Corporativo (Power BI)

## Objetivo
Estruturar um painel para análise de consumo de telefonia corporativa,
permitindo identificar padrões de uso, linhas ociosas e oportunidades
de redução de custos.

## Contexto do Projeto
Projeto demonstrativo utilizando dados fictícios estruturados para
simular arquivos reais de bilhetagem de operadoras.

## Escopo Analítico
- Consumo por linha
- Consumo por centro de custo
- Tipo de uso (voz, dados, roaming)
- Análise temporal

## Principais Indicadores
- Consumo total
- Consumo médio por linha
- Linhas subutilizadas
- Picos de uso
- Distribuição por centro de custo

## Ferramentas Utilizadas
- Power BI Desktop
- Power Query (M)
- Arquivos CSV / Excel

## Observações
Os dados utilizados são totalmente fictícios e não representam
informações reais de operadoras ou empresas.
